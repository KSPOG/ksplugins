# KSP Plugins Landing Page

Static, responsive landing page for GitHub Pages, Cloudflare Pages, Netlify, or any basic host.

## Before publishing

Open `index.html` and replace:

- `https://discord.gg/REPLACE_ME`
- `REPLACE_WITH_SUPPORT_EMAIL`
- `https://REPLACE_WITH_TWILITE_STORE_URL`
- Product names, versions, descriptions, and store links

Search the folder for `REPLACE_` before publishing.

## Preview locally

```powershell
py -m http.server 8080
```

Open `http://127.0.0.1:8080`.

## GitHub Pages

1. Create a public repository.
2. Upload all files from this folder to the repository root.
3. Open **Settings → Pages**.
4. Choose **Deploy from a branch**.
5. Select `main` and `/ (root)`.
6. Save and use the public URL GitHub provides.

## Review checklist

- No placeholder links remain
- Contact method works
- Products and automation features are described accurately
- Terms, Privacy, and Refund pages are accessible
- Website loads without login

The legal pages are practical templates, not jurisdiction-specific legal advice.
